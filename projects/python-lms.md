This project was developed for a sponsor who wanted an accessible platform to teach young students the basics of Python programming. I designed the frontend using Bootstrap and implemented Firebase as the backend storage solution.

Beyond the technical work, I took on a leadership role — organizing and leading monthly meetings with two teammates and the sponsor to present progress, gather feedback, and incorporate new requests into the development roadmap. This project gave me experience balancing client communication alongside hands-on software development.

[Here](https://github.com/jeff-yc-wong/python-lms) is a link to the project on Github.
