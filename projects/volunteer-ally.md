Volunteer Ally is a web application designed to better connect volunteers with the organizations that need them. The platform displays relevant information tailored to volunteer work and facilitates direct communication between volunteers and the appropriate organizational contacts.

I collaborated with four teammates to build and test the application using the Meteor framework for the full-stack web layer and MongoDB for storing user and organizational data. We used GitHub for version control and coordinated code contributions across the team throughout the development lifecycle.
